# 🍠겨울 간식 지도🍠
##### 겨울의 대표 간식인 붕어빵과 군고구마의 위치를 찾기 쉽게 지도로 제공하는 웹 서비스
###### 🌟 <ins>웹 크롤링</ins>을 통해 수집한 서울시 각 구에서 판매되는 겨울 간식의 주소를 지도상에 표시
###### 🌟 각 구별로 겨울 간식의 수량을 아이콘 크기로 직관적으로 표시하고, 클릭 시 상세 위치를 제공
